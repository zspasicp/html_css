U prvoj lekciji je objasnjena osnovna struktura HTML i CSS fajlova.
Instrukcije su napisane u vidu komentara:
HTML komentar: <!-- Ovo je html komentar -->
CSS komentar: /* Ovo je css komentar */

kada browser naidje na komentar, jednostavno ga preskoci. Komentari sluze samo da bi developeri lakse pratili kod.

Kada instaliras atom, izaberi opciju load folder (ili tako nesto) i izaberi ovaj Lesson 1 folder, i on ce ti otvoriti sve fajlove koji
se nalaze u ovom folderu, i imaces tagove obojene bojama za lakse snalazenje. dobra stvar kod njega je sto recimo ukucas samo div i udaris enter
i on ti sam napise kako taj tag treba da izgleda.

aplikaciju pokreces tako sto pokrenes index.html pomocu browsera (mozzila je bolja za razvoj jer chrome ponekad ne dozvoljava pokretanje aplikacija sa lokalnog sistema)
